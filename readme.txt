== Description ==
This is a Payment plugin that provides a points by Authorize.net which allows user buy points for purchase. To use this plugin just use the shortcode [cp_authtopupcp].

Features are as follows:

* Shortcodes for displaying the form [cp_authtopupcp]
* Settings screen for adjusting various settings

There are css styles that you can override in your own template.

I'm also available for paid support only should you need any.

== Installation ==

   1 Upload cubepoints-authorize.net.zip to the /wp-content/plugins/ directory
   2 Extract the zip file (cubepoints-authorize.net.zip)
   3 Activate the plugin through the 'Plugins' menu in WordPress
   4 Place shortcode [cp_authtopupcp] anywhere in the posts or pages to display the form
